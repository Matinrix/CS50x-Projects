## 💜 About This Repository
Welcome to **Week 3** of Harvard’s _CS50_ course!  
In this week, we shift from “how to write code” to **thinking algorithmically**:  
what an algorithm is, how to categorize algorithms, and how to **organize data efficiently** using classic sorting/search techniques.

> 🧭 These topics build the mental model you need before moving into more advanced data structures.

- **Compiler:** GCC / clang

## 🧠 Concepts Covered
### 1) `typedef struct` 
- Defining custom data types
- Storing related fields in one structure
- Using structs to model real-world entities

### 2) What is an Algorithm?
- A step-by-step method to solve a problem
- Why algorithms matter: correctness + efficiency

### 3) Algorithm Types & Thinking
- Conceptual differences between approaches
- When to choose one strategy over another

### 4) Sorting 
- Sorting as organizing information for faster/cleaner searching
- Intro to approaches like:
  - **Bubble Sort** (and related basic sorting ideas)
